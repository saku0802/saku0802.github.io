SNS管理ツール「Twittersskills」開発

プロジェクトの概要

10名規模の開発チームを率い、SNS管理ツール「Twittersskills」を開発しました。
Twitter APIを活用し、投稿の自動化、フォロワー分析、効果測定機能などを実装しました。
その結果、リリース後1ヶ月で100社以上の企業に導入されるなど、大きな成果を収めました。

使用技術

  • 言語: Python, JavaScript
  • フレームワーク: Django, React
  • データベース: PostgreSQL
  • その他: AWS, Docker, CircleCI

役割

プロジェクトマネージャーとして、要件定義、設計、開発、テスト、リリース、運用まで、プロジェクト全体の統括を担当しました。
また、エンジニアチームのマネジメント、進捗管理、品質管理なども行いました。

成果

  • 目標期限を1ヶ月前倒しで達成
  • リリース後1ヶ月で100社以上の企業に導入
  • 顧客満足度95%を獲得
